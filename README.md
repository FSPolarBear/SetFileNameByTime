# SetFileNameByTime
依据文件最后修改时间将文件按顺序重命名为1.xxx、2.xxx、……，不改变文件的扩展名

## 版本
1.0.0

## 环境要求
Windows10, .NET 6.0

## 使用方法
本程序为命令行程序，需要两个参数。<br>
参数1 源地址：需要重命名的文件路径<br>
参数2 目标地址：重命名后文件所在路径

## 注意事项
该程序会重命名源地址下的所有文件，但不会对源地址的子文件夹内的内容进行操作。<br>
重命名后保留源地址的文件。<br>
如源地址或目标地址不存在，或目标地址中已存在与被重命名后的文件同名的文件，会导致重命名失败。<br>
